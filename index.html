What is the RFC?

The RFC (Federal Taxpayers Registry) is a code issued by Mexico’s Tax Administration Service (SAT) to identify individuals and companies for tax purposes. It’s typically a combination of letters and numbers derived from a person’s name and birth date. It’s essential for any official documentation or financial activity in Mexico.

Why is the RFC Important?

For anyone working, opening a business, or needing official identification in Mexico, the RFC is essential. It’s used for tax purposes but is also required for other official tasks, such as opening a bank account. This unique identifier helps the government keep track of tax obligations and other legal matters.

RFC Structure

Before diving into calculating an RFC without a paternal surname, let’s review the general RFC structure:

  • First Four Letters: Derived from the name (first letter of the paternal surname, first vowel of the paternal surname, first letter of the maternal surname, and first letter of the first name).
  • Six Digits: Represent the birth date in the format YYMMDD.
  • Homoclave: A unique three-digit code to avoid duplication.

How to Calculate the RFC for Someone Without a Paternal Surname

When there’s no paternal surname, the process for generating the RFC changes slightly. Here’s how it works step-by-step:

1. Substitute the Paternal Surname Section

For people without a paternal surname, SAT rules specify using the first two letters of the maternal surname instead. This ensures the RFC maintains the correct length.

2. Use the First Name

Then, use the first letter of the first name. For example, if the name is "Maria Garcia," the initial part of the RFC would be "GAM."

3. Add the Birth Date

The next part of the RFC is generated with the birth date in the format YYMMDD. For example, if the birth date is April 3, 1995, "950403" would be added.

4. Calculate the Homoclave

The homoclave is the final part of the RFC and consists of three additional characters assigned by the SAT. This is unique for each person, so it doesn’t need to be calculated manually.
